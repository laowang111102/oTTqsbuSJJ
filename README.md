# 宿舍管理系统简介 python294

本项目是基于 Python 的期末大作业，旨在为学校宿舍管理提供高效、便捷的解决方案。

## 技术栈

- Python
- 前端框架（如：Flask、Django）
- 数据库（如：MySQL、SQLite）

## 功能模块

### 学生管理

- 添加、修改和删除学生信息
- 分配和调整学生宿舍
- 按条件搜索学生
  - 学号
  - 姓名
  - 学院
- 查看学生详细信息与宿舍分配情况

### 宿舍管理

- 添加、修改和删除宿舍楼和宿舍信息
- 管理宿舍容量和入住情况
- 查看宿舍状态
  - 空
  - 已住满
  - 部分入住
  - 维修
- 宿舍分配和调整功能

### 报修管理

- 记录宿舍报修信息
  - 宿舍号
  - 问题描述
  - 提交日期
- 处理报修请求
  - 分配维修人员
  - 记录处理结果
- 跟踪报修状态
  - 待处理
  - 处理中
  - 已完成
- 统计分析报修数据

### 考勤管理

- 记录学生异常情况
  - 晚归
  - 未归
- 搜索考勤记录
  - 学号
  - 日期
  - 类型
- 修改和删除考勤记录
- 统计分析考勤数据

### 费用管理

- 添加不同类型的费用
  - 住宿费
  - 水电费
- 标记费用支付状态
- 查看费用详情和支付记录
- 统计分析费用数据

### 数据统计

- 学生统计
  - 总数
  - 性别分布
  - 学院分布
- 宿舍统计
  - 总数
  - 状态分布
  - 入住率
- 报修统计
  - 总数
  - 处理效率
  - 高频宿舍
- 费用统计
  - 总费用
  - 支付率
  - 费用类型分布

## 系统角色

- 学生
- 管理员
- 维修人员

## 运行环境

- Python 环境
- 前端框架所需环境
- 数据库服务
